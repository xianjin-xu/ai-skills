# Switch

## 概述

Switch 组件

> Switch Component

**分类**: 表单输入
**在线演示**: [https://www.blazor.zone/switch](https://www.blazor.zone/switch)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `OnColor` | `Color` | `Color.Success` | 获得/设置 开颜色 |
| `OffColor` | `Color` | `}` | 获得/设置 关颜色 |
| `int` | `override` | `40` | 获得/设置 组件宽度 默认 40 |
| `Height` | `int` | `20` | 获得/设置 控件高度默认 20px |
| `OnInnerText` | `string?` | `}` | 获得/设置 组件 On 时内置显示文本 |
| `OffInnerText` | `string?` | `}` | 获得/设置 组件 Off 时内置显示文本 |
| `ShowInnerText` | `bool` | `}` | 获得/设置 是否显示内置文字 默认 false 显示 |
