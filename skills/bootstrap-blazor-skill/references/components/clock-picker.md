# ClockPicker

## 概述

ClockPicker 组件

> ClockPicker component

**分类**: 表单输入
**在线演示**: [https://www.blazor.zone/clock-picker](https://www.blazor.zone/clock-picker)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `ShowClockScale` | `bool` | `}` | 获得/设置 是否显示表盘刻度 默认 false |
| `ShowSecond` | `bool` | `true` | 获得/设置 是否显示秒 默认 true |
| `ShowMinute` | `bool` | `true` | 获得/设置 是否显示分钟 默认 true |
| `IsAutoSwitch` | `bool` | `true` | 获得/设置 是否自动切换 小时、分钟、秒 自动切换 默认 true |
