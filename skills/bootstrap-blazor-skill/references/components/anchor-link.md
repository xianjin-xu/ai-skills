# AnchorLink

## 概述

AnchorLink 组件

> AnchorLink component

**分类**: 布局
**在线演示**: [https://www.blazor.zone/anchor-link](https://www.blazor.zone/anchor-link)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `Text` | `string?` | `}` | 获得/设置 组件 Text 显示文字 |
| `TooltipText` | `string?` | `}` | 获得/设置 组件 拷贝成功后 显示文字 |
| `Icon` | `string?` | `}` | 获得/设置 锚点图标 默认 fa-solid fa-link |
