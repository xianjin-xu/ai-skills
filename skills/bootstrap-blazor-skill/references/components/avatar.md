# Avatar

## 概述

Avatar 头像框组件

> Avatar component

**分类**: 数据展示
**在线演示**: [https://www.blazor.zone/avatar](https://www.blazor.zone/avatar)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `IsCircle` | `bool` | `}` | 获得/设置 是否为圆形 |
| `Url` | `string?` | `}` | 获得/设置 Image 头像路径地址 |
| `IsIcon` | `bool` | `}` | 获得/设置 是否为图标 |
| `Icon` | `string?` | `}` | 获得/设置 头像框显示图标 |
| `IsText` | `bool` | `}` | 获得/设置 是否显示为文字 |
| `Text` | `string?` | `}` | 获得/设置 头像框显示文字 |
| `Size` | `Size` | `Size.Medium` | 获得/设置 头像框大小 |
| `IsBorder` | `bool` | `}` | 获得/设置 是否显示 Border 默认为 false |
| `GetUrlAsync` | `Func<Task<string>>?` | `}` | 获得/设置 获取图片地址异步回调方法 |
