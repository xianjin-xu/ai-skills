# Marquee

## 概述

Marquee 字幕滚动组件

> Marquee scrolling component

**分类**: 数据展示
**在线演示**: [https://www.blazor.zone/marquee](https://www.blazor.zone/marquee)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `Text` | `string?` | `}` | 获得/设置 组件值 显示文本 默认 Empty |
| `Color` | `string` | `"#000"` | 获得/设置 组件值 文本颜色 默认 #000 支持16进制和颜色名称 |
| `BackgroundColor` | `string` | `"#fff"` | 获得/设置 组件值 背景颜色 默认 #fff 支持16进制和颜色名称 |
| `FontSize` | `int` | `72` | 获得/设置 组件值 文本大小 默认 72px |
| `Duration` | `int` | `14` | 获得/设置 组件值 动画时间 默认 14s <para>值越小滚动越快 |
| `Direction` | `MarqueeDirection` | `}` | 获得/设置 组件值 滚动方向 默认 LeftToRight |
