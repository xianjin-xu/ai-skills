# Progress

## 概述

Progress 组件

> Progress Component

**分类**: 数据展示
**在线演示**: [https://www.blazor.zone/progress](https://www.blazor.zone/progress)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `Height` | `int?` | `}` | 获得/设置 控件高度，默认为 null 未设置 |
| `Color` | `Color` | `Color.Primary` | 获得/设置 颜色，默认为 Color.Primary |
| `IsShowValue` | `bool` | `}` | - |
| `IsStriped` | `bool` | `}` | - |
| `IsAnimated` | `bool` | `}` | - |
| `Value` | `double` | `}` | 获得/设置 组件进度值 |
| `Round` | `int` | `}` | 获得/设置 进度值修约小数位数，默认为 0（即保留为整数） |
| `MidpointRounding` | `MidpointRounding` | `MidpointRounding.AwayFromZero` | 获得/设置 保留小数点模式，默认为 AwayFromZero |
| `Text` | `string?` | `}` | 获得/设置 进度标签文本 |
