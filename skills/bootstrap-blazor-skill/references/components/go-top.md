# GoTop

## 概述

GoTop 组件

> GoTop Component

**分类**: 布局
**在线演示**: [https://www.blazor.zone/go-top](https://www.blazor.zone/go-top)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `Icon` | `string?` | `}` | 获得/设置 返回顶端 Icon 属性 |
| `Target` | `string?` | `}` | 获得/设置 滚动条所在组件 |
| `ScrollBehavior` | `ScrollIntoViewBehavior` | `}` | 获得/设置 滚动行为 默认 ScrollIntoViewBehavior.Smooth |
| `TooltipText` | `string?` | `}` | 获得/设置 鼠标悬停提示文字信息 |
