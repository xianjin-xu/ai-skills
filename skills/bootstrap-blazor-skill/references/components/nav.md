# Nav

## 概述

NavMenu 组件基类

> NavMenu Component Base Class

**分类**: 导航

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `Items` | `IEnumerable<NavLink>?` | `}` | 获得/设置 组件数据源 |
| `Alignment` | `Alignment` | `Alignment.Left` | 获得/设置 组件对齐方式 |
| `IsVertical` | `bool` | `}` | 获得/设置 是否垂直分布 |
| `IsPills` | `bool` | `}` | 获得/设置 是否为胶囊 |
| `IsFill` | `bool` | `}` | 获得/设置 是否填充 |
| `IsJustified` | `bool` | `}` | 获得/设置 是否等宽 |
| `ChildContent` | `RenderFragment?` | `}` | 获得/设置 组件内容 |
