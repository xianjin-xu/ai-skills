# Anchor

## 概述

Anchor 组件部分类

> Anchor component

**分类**: 布局
**在线演示**: [https://www.blazor.zone/anchor](https://www.blazor.zone/anchor)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `Target` | `string?` | `}` | 获得/设置 目标组件 Id |
| `Container` | `string?` | `}` | 获得/设置 滚动组件 Id 默认为 null 使用最近滚动条容器元素 |
| `IsAnimation` | `bool` | `true` | 获得/设置 滚动时是否开启动画 默认 true |
| `Offset` | `int` | `}` | 获得/设置 距离顶端偏移量 默认为 0 |
| `ChildContent` | `RenderFragment?` | `}` | 获得/设置 子内容 |
