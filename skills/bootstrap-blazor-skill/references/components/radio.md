# Radio

## 概述

Radio 单选框组件

> Radio Component

**分类**: 表单输入
**在线演示**: [https://www.blazor.zone/radio](https://www.blazor.zone/radio)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `Task` | `Func<TValue,` | `}` | 获得/设置 点击回调方法 |
| `GroupName` | `string?` | `}` | 获得/设置 Radio 组名称。一般来讲需要设置，默认为 null 未设置 |
