# Spinner

## 概述

Spinner 组件基类

> Spinner Component

**分类**: 数据展示
**在线演示**: [https://www.blazor.zone/spinner](https://www.blazor.zone/spinner)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `Color` | `Color` | `}` | 获得/设置 Spinner 颜色 默认 None 无设置 |
| `Size` | `Size` | `}` | 获得 / 设置 Spinner 大小 默认 None 无设置 |
| `SpinnerType` | `SpinnerType` | `}` | 获得/设置 Spinner 类型 默认为 Border |
