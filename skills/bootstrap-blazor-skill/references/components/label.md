# Label

## 概述

BootstrapLabel 组件

> BootstrapLabel component

**分类**: 基础组件
**在线演示**: [https://www.blazor.zone/label](https://www.blazor.zone/label)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `Value` | `string?` | `}` | 获得/设置 组件值 显示文本 默认 null |
| `ShowLabelTooltip` | `bool?` | `}` | 获得/设置 是否显示 Tooltip 多用于标签文字过长导致裁减时使用 默认 false 不显示 |
| `LabelWidth` | `int?` | `}` | 获得/设置 标签宽度 默认 null 未设置使用全局设置 <code>--bb-row-label-width</code> 值 |
