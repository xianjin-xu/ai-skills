# NetworkMonitor

## 概述

NetworkMonitorIndicator 组件用于显示网络监控指示器，支持自定义弹出窗口设置

> Represents a network monitor indicator with customizable tooltip settings

**分类**: 其他
**在线演示**: [https://www.blazor.zone/network-monitor](https://www.blazor.zone/network-monitor)

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `Title` | `string?` | `}` | 获得/设置 Popover 弹窗标题，默认为 null |
| `PopoverPlacement` | `Placement` | `Placement.Top` | 获得/设置 Popover 显示位置，默认为 Top |
| `Trigger` | `string?` | `}` | 获得/设置 Popover 触发方式，默认为 hover focus |
