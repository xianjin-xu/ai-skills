# Filters

## 概述

BoolFilter 组件用于表格列的布尔值过滤

> BoolFilter component is used for boolean value filtering in table column

**分类**: 其他

## 参数 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `Items` | `IEnumerable<SelectedItem>?` | `}` | 获得/设置过滤候选项，建议使用静态数据以避免性能损失 |
